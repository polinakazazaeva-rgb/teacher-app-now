## 📖 Описание

**"Справочник преподавателя"** — это консольное приложение с графическим интерфейсом (Tkinter), разработанное для преподавателей дисциплины.

Приложение позволяет:
- Хранить программу курса 
- Фиксировать планируемые результаты обучения
- Изучать методики преподавания
- Вести расписание занятий
- Хранить личные заметки
- Ведение журнала успеваемости с подсчётом средней оценки
- Отслеживание должников

Проект написан в учебных целях.

## ⚡ Функциональные возможности

| Функция | Описание |
|---------|----------|
| 🔐 **Авторизация** | Регистрация и вход с проверкой надёжности пароля |
| 📖 **Содержание программы** | Полная программа курса (68 часов, 7 разделов) |

| 📋 **Методики обучения** | 6 педагогических методик с примерами |
| 📅 **Расписание** | Расписание занятий + загрузка из Excel |
| 📝 **Мои заметки** | Личные заметки преподавателя |
| 📊 **Журнал успеваемости** | Ученики, оценки, должники, средний балл |

## 🛠 Технологии

- **Python 3.8+** — язык программирования
- **Tkinter** — графический интерфейс
- **JSON** — файловое хранилище данных
- **Openpyxl (опционально)** — для загрузки расписания из Excel

## 🚀 Установка и запуск

### 1. Скопируйте проект

```bash
git clone https://github.com/ваш-ник/teacher_app.git
cd teacher_app

2. Установите зависимости (опционально)
Для работы загрузки из Excel:

bash
pip install openpyxl
3. Запустите приложение
bash
python "teacher app.py"

4. Первый вход
Нажмите "СОЗДАТЬ АККАУНТ"

Заполните: логин, имя, пароль (минимум 6 символов, заглавная буква и цифра)

Подтвердите пароль

Войдите в систему

📁 Структура проекта
text
teacher_app/
│
├── teacher app.py          # Главный файл приложения
├── spravochnik_data.json   # Данные пользователей и заметки
├── students.json           # Данные учеников и оценки
└── README.md               # Документация
📖 Инструкция по использованию
🔐 Вход в систему
Введите логин и пароль

При первом входе создайте аккаунт через кнопку "СОЗДАТЬ АККАУНТ"

📖 Раздел "Содержание программы"
Полная программа курса по информационным технологиям (68 часов)

7 основных разделов с указанием часов

🎯 "Планируемые результаты"
Цели и задачи курса

Внесённые изменения в программу

Количество часов (лекции + практика)

📋 "Методики обучения"
6 методик: метод проектов, проблемное обучение, кейс-метод, игровые технологии, перевёрнутый класс, дифференцированное обучение

Суть и примеры применения

📅 "Расписание"
Встроенное расписание занятий по дням

Функция: загрузить расписание из Excel-файла (первые два столбца)

📝 "Мои заметки"
Личное пространство для заметок преподавателя

Сохраняются в профиле пользователя

📊 "Журнал успеваемости"
Включает 4 вкладки:

Вкладка	Функция
Ученики	Добавление / удаление учеников
Оценки	Выбор ученика → тема → оценка
Должники	Автоматическое отображение учеников с оценками 1 или 2
Средний балл	Общая статистика и средний балл каждого ученика
📂 Файлы данных
Приложение автоматически создаёт два JSON-файла:

Файл	Содержимое
spravochnik_data.json	Пользователи: логин, имя, пароль, заметки
students.json	Ученики: список с оценками по темам
Файлы сохраняются в той же папке, что и программа.

🎓 Для студентов
Проект написан без использования классов (ООП), что делает его идеальным для изучения:

✅ Работа с Tkinter (виджеты, события, макеты)

✅ Работа с JSON (чтение и запись данных)

✅ Обработка ошибок и валидация

✅ Проектирование GUI-приложений

Основные функции для изучения:

load_data() / save_data() — работа с файлами

check_password_strength() — валидация

show_journal() — сложный интерфейс с вкладками

refresh_*() — обновление интерфейса
